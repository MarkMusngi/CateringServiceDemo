The De Florence Catering Web Application is a specialized platform designed to streamline catering operations. It features a robust admin dashboard for secure management and efficient handling of complex order data.

Features
Admin Dashboard: A secure management interface for overseeing catering operations.
Order Management: Handles complex order data through dedicated backend logic.
Full CRUD Functionality: Allows for creating, reading, updating, and deleting records within the system.
RESTful API: Facilitates efficient communication between the frontend and the server.

Technical Stack
Backend: Node.js and Express.js.
Architecture: RESTful APIs.
Database Management: Experienced in managing Relational and NoSQL databases.
Development Tools: Managed using Git, GitHub, and Jira.
