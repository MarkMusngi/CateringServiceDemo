# Catering Services Management System

A professional event management and booking platform designed for catering businesses. This system allows administrators to handle the end-to-end lifecycle of a catering event, from client registration to menu customization and final status tracking.

***

##  Key Features

* **Admin Dashboard:** A centralized hub to view, filter, search, and sort catering events by date, name, or status.
* **Client Profiling:** Dedicated section to manage client contact information and history.
* **Comprehensive Event Forms:** Manage every detail of an event:
    * **Menu Planning:** Detailed categories for Welcome Food, Appetizers, Main Courses (up to 7 items), Desserts, and Drinks.
    * **Logistics:** Track Venue, Address, Occasion, and Table Setup (Linens/Napkins).
    * **Financials:** Manage Downpayments and Balances.
* **Email Automation:** Integrated **Nodemailer** functionality to send official "Confirmation Emails" once an event status is set to 'Confirmed'.
* **Dynamic Search:** Real-time client search with debouncing for a smooth administrative experience.



***

## Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** MongoDB (using MongoDB Driver)
* **Templating Engine:** Handlebars (HBS)
* **Styling:** Bootstrap 5 & Custom CSS
* **Communication:** Nodemailer (SMTP)
